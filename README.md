TechEvents Portal
Overview

TechEvents Portal is a web-based platform designed to manage and participate in college technical events. It provides an interface for both students and administrators to interact with event-related activities such as browsing, registering, posting, and analyzing events.

Features
Student Features
Login using student credentials
View upcoming and categorized events
Search events by name
Register for events
View registered events in dashboard
Participate in event discussions
Admin Features
Login using admin credentials
Create, edit, and delete events
Upload images and documents for events
Manage event discussions
View registration analytics with department-wise breakdown
General Features
Responsive homepage with sections like About and How It Works
Smooth navigation across multiple views
Real-time countdown for event registration deadlines
Search functionality for quick event access
Contact section for department details
Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Google Fonts (Poppins)
Project Structure
Header with navigation and search bar
Login and Registration interface
Homepage with hero section and information sections
Event browsing and category selection
Dashboard for students and admins
Event posting and management system
Contact section for departments
How It Works
User logs in as Student or Admin
Student can explore events and register
Admin can create and manage events
Registered events are stored and displayed in dashboard
Admin dashboard shows analytics of registrations
Key Functionalities
Dynamic event rendering using JavaScript
Event registration tracking
Department-wise analytics generation
Interactive UI with animations and transitions
Comment/discussion system for events
Usage
Open the HTML file in a web browser
Login using any credentials (no backend validation implemented)
Navigate through different sections using the menu
Register for events or manage events based on role
Limitations
No backend or database integration
Data is not persistent (resets on page refresh)
Authentication is only simulated
Future Enhancements
Integration with backend (Node.js / Flask / Django)
Database support (MySQL / MongoDB)
Real authentication system
Email notifications for registrations
Improved UI/UX and mobile responsiveness
